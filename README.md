# create-react-app template using TypeScript and Redux (toolkit)

## Features

- [React](https://reactjs.org/) - JavaScript UI library
- [TypeScript](https://www.typescriptlang.org/) - Type safe superset of JavaScript
- [Redux (toolkit)](https://redux-toolkit.js.org/) - State management library for JavaScript

```
mkdir my-app
cd my-app
git init
git pull https://github.com/KresimirCosic/cra-ts-redux-template
npm install
```

## Usage

> I don't install packages globally, so I usually run <strong>npx npm start</strong> to start the application. The application should start using self-signed certificate so expect a warning before it's added to the trusted certificates.
